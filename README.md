# Chaos Simulator

Chaos Simualtor is a WEB game about Chaos made with Laravel and VueJS

## Dev

### 1. Copy .env file

```bash
cp .env.example .env
```

### 2. Download dependencies

```bash
composer install
```

### 3. Generate key

```bash
php artisan key:generate
```

### 4. Downaload javascript dependencies

```bash
npm install
```

### 5. Launch server

Terminal :

```bash
npm run start
```
